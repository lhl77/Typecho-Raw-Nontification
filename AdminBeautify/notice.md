感谢你安装 **Admin Beautify** 插件！这里是一份简要使用指南。

### 🎨 基础设置

前往 **控制台 → 插件 → Admin Beautify → 设置** 即可打开插件配置页面，按需调整：

- **主题色** — 7 种色调自由切换（紫 / 蓝 / 青 / 绿 / 橙 / 粉 / 红）
- **暗色模式** — 亮色 / 暗色 / 跟随系统
- **登录页美化** — 12+ 配色预设，支持自定义背景图与虚化
- **AJAX 导航** — 开启后后台页面无刷新切换
- **编辑器增强** — 可选启用 Vditor 富文本 / Markdown 编辑器
- **PWA** — 将博客后台安装为桌面 / 手机应用

### 🛠 常见问题

**样式异常？** 清除浏览器缓存（Ctrl+Shift+R）后刷新。

**Vditor 编辑器不显示？** 确认在设置中已启用"编辑器增强"，并检查网络是否可访问 CDN。

**PWA 安装后打不开？** 在设置中点击"重新生成 Service Worker"。
